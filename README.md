# Superstore Sales & Customer Analytics Dashboard

End-to-end sales and customer analytics project using Excel, SQL, Python, and Power BI

## Executive Summary

This project analyzes Superstore retail sales data to evaluate sales performance, customer behavior, and operational efficiency. Using SQL for data analysis, Python for exploratory validation, and Power BI for visualization, the project identifies revenue drivers, seasonal trends, customer retention patterns, and regional delivery performance to support data-driven business decisions.

## Tools & Technologies

- SQL (PostgreSQL) – data aggregation and analysis  
- Python (Pandas) – exploratory analysis and validation
- Microsoft Excel & Power Query – data cleaning, transformation, and feature creation
- Power BI – interactive dashboard and visualization  

## Business Questions Answered

- How do sales trend across months and seasons?
- Which regions and products contribute the most to revenue?
- What percentage of customers are repeat vs one-time?
- How does delivery performance vary by region?

## Key Insights

- Consumer segment contributes ~50% of total revenue  
- Sales peak in Q4, indicating strong seasonality  
- Repeat customers dominate across all segments  
- West region shows higher average delivery time  

## Dashboard Features

- KPI cards for total sales, total orders, and average delivery time  
- Monthly sales trend analysis  
- Regional sales visualization using maps  
- Top 10 products by revenue  
- Customer retention analysis by segment  
- Interactive filters for year, region, and segment
